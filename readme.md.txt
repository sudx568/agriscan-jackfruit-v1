# 🍈 Jackfruit Quality Checker (AI-Powered)
**Developed by Sudarshan | BCA AI Project**

### 🌟 Overview
A Deep Learning-based web application designed to assess the quality of Jackfruits. Using **MobileNetV2**, the system classifies fruit as **Grade A (Good)** or **Grade C (Bad)** with high precision.

### 🚀 Key Features
* **Deep Learning:** Built with TensorFlow/Keras and MobileNetV2.
* **Multi-Lingual:** Supports Kannada, Tamil, Telugu, Malayalam, Hindi, and English.
* **Real-time Scanning:** Interactive UI with scan-line animations.
* **Lightweight:** Optimized for speed and low-power devices.

### 🛠️ Tech Stack
* **Language:** Python 3.10
* **Framework:** Flask
* **Frontend:** Tailwind CSS
* **AI Model:** Keras (Transfer Learning)

### 📖 How to Run
1. Clone the repo: `git clone [your-link]`
2. Activate venv: `.\venv\Scripts\activate`
3. Install deps: `pip install -r requirements.txt`
4. Run: `python app.py`
